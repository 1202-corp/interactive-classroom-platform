# Interactive Classroom Platform

> [Русская версия](docs/README-RU.md)

A web application for conducting interactive classes in university auditoriums. The platform increases student engagement during lectures and seminars by using smartphones and laptops for real-time feedback.

## Overview

Interactive Classroom Platform consists of several components that work together to provide an interactive learning experience:

- **Frontend**: Web application for teachers, students, and projector displays
- **Backend API**: Server-side logic, session management, real-time communication
- **Telegram Bot**: Integration with group chats for sending session invitations

## Key Features

- Session creation and QR code connection
- Q&A mode: collecting questions from students and answers to teacher's questions
- Quizzes: conducting tests with multiple choice answers and displaying results
- General chat: text chat for free discussion (optional)
- Reactions: emoji feedback for quick audience response
- Activity rating: tracking and rewarding the most active participants

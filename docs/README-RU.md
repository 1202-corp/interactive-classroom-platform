# Документация платформы

> [English version](../README.md)

Основная документация платформы интерактивных занятий Interactive Classroom Platform.

## Описание

Interactive Classroom Platform – веб-приложение для проведения интерактивных занятий в аудиториях университета. Платформа повышает вовлеченность студентов во время лекций и семинаров за счет использования смартфонов и ноутбуков для обратной связи в реальном времени.

## Основные возможности

- **Создание сессий и подключение по QR-коду**: Преподаватель создает сессию, система генерирует QR-код, студенты подключаются через браузер
- **Режим Q&A**: Сбор вопросов от студентов и ответов на вопросы преподавателя с системой голосования
- **Викторины**: Проведение тестов с вариантами ответов и отображением результатов
- **Общий чат**: Текстовый чат для свободного обсуждения (опционально)
- **Реакции**: Эмодзи-отклики для быстрой обратной связи от аудитории
- **Рейтинг активности**: Отслеживание и поощрение самых активных участников

## Компоненты платформы

- **Frontend**: Веб-приложение для преподавателей, студентов и проектора
- **Backend API**: Серверная логика, управление сессиями, real-time коммуникация
- **Telegram-бот**: Интеграция с групповыми чатами для рассылки приглашений

## Целевая аудитория

- Преподаватели и лекторы вузов
- Студенты
- Организаторы образовательных мероприятий

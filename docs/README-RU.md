# Interactive Classroom Platform

> [English version](../README.md)

Веб-приложение для проведения интерактивных занятий в аудиториях университета. Платформа повышает вовлеченность студентов во время лекций и семинаров за счет использования смартфонов и ноутбуков для обратной связи в реальном времени.

## Обзор

Interactive Classroom Platform состоит из нескольких компонентов, которые работают вместе для обеспечения интерактивного обучения:

- **Frontend**: Веб-приложение для преподавателей, студентов и проектора
- **Backend API**: Серверная логика, управление сессиями, коммуникация в реальном времени
- **Telegram Bot**: Интеграция с групповыми чатами для рассылки приглашений на сессии

## Ключевые функции

- Создание сессий и подключение по QR-коду
- Режим Q&A: сбор вопросов от студентов и ответов на вопросы преподавателя
- Викторины: проведение тестов с вариантами ответов и отображение результатов
- Общий чат: текстовый чат для свободного обсуждения (опционально)
- Реакции: эмодзи-отклики для быстрой обратной связи от аудитории
- Рейтинг активности: отслеживание и поощрение самых активных участников
